## 1. 项目概述

### 目标与背景
你在 Windows / macOS 来回换电脑时，经常需要同步：
- 代码项目
- Codex 配置和聊天记录
- 额外技能（`.agents`）

手动复制很容易漏文件、路径不一致、还容易复制一堆没用的“编译产物”。AgentSync 的目标是让这件事变成“一键搞定”。

### 范围
- **范围内（MVP）:**
  - 单向同步：源电脑 → 目标电脑
  - 传输方式：SSH（SFTP）
  - 可选择同步项目文件夹 / Codex / `.agents`
  - 默认镜像删除 + 自动备份（避免误删不可恢复）
  - 可查看同步日志和历史
- **范围外（先不做）:**
  - 双向同步与冲突合并
  - 云同步（网盘/对象存储）
  - 企业级权限/多用户

### 干系人
- **负责人:** 你（本仓库的使用者）
