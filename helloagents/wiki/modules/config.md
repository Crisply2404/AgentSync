# Config

## 目的
把用户在界面里填的东西保存下来，下次打开还能用。

## 模块概述
- **职责:** 配置读写、默认路径识别、基础校验
- **状态:** 🚧开发中
- **最后更新:** 2026-02-14

## 规范

### 需求: 自动填默认路径
**模块:** Config  
在不同系统上自动给出常见路径，减少手动输入。

#### 场景: Windows
前置条件：系统为 Windows  
- 预期结果：Codex 默认路径建议为 `%USERPROFILE%\\.codex\\`

#### 场景: macOS / Linux
前置条件：系统为 macOS 或 Linux  
- 预期结果：Codex 默认路径建议为 `~/.codex/`

### 需求: 管理 AgentSync 专用 SSH 密钥
**模块:** Config / SSH Keys  
为了让“设备向导”一键跑通，需要一个默认的、可复用的专用密钥位置。

#### 场景: 第一次生成密钥
前置条件：`~/.agentsync/keys/agentsync_ed25519` 不存在  
- 预期结果：自动创建目录并生成密钥；配置里只保存“私钥路径”，不保存私钥内容
